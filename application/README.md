In this repo you will find both the front and backend code (all in JS)

## Clone 

```bash
git clone https://github.com/drbh/asu-tln-react-poc.git
cd asu-tln-react-poc
```

# Frontend

## Install front end deps

```bash
yarn install
```

## Start frontend

```bash
yarn start
```

## Build frontend

```bash
yarn build
```

# Backend

```bash
cd server
```

## Start backend

```bash
node server.js
```
